# Numlock on startup

It infuriates me that the numlock is not activated by default and that Linux doesn't remember that numlock was activated before shutting down the computer. 

And I never saw in the install process how to activate it.

For the moment there will be only for KDE because that's the desktop environment I use for the moment. 

## Step
### KDE
- Open System Settings > Keyboard.
- Set NumLock on Startup to Turn on.
- Go to Colors and Appearance > Login Screen (SDDM).
- Click Apply Plasma Settings... at the top right to make it apply to the login screen.

I didn't know you needed to apply the settings for the login screen until today (2026-03-22). Not really intuitive if you want my opinion.
